# Bosnia Lens Frontend

React frontend application for Bosnia Lens - a web portal for exploring open data about Bosnia and Herzegovina including cities, zip codes, holidays, and more.

## Features

- React 19 with modern hooks
- Vite for fast development and building
- Tailwind CSS for styling
- React Router for navigation
- JavaScript (ES modules)
- ESLint configuration
- Testing setup with Vitest and React Testing Library

## Installation

```bash
# Clone the repository
git clone git@github.com:goran1010/bosnia-lens.git

# Navigate to project directory
cd bosnia-lens/frontend

# Install dependencies
npm install

# Rename the environment file
cp .env.example .env
# Edit .env file as needed
```

## Usage

```bash
# Development server
npm run dev

# Build for production
npm run build

# Run tests
npm run test
```
